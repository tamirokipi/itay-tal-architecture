## What They Need
An AI copywriting agent that integrates with their website builder to auto-generate professional Hebrew landing page copy for their clients.

### The Problem
- Every landing page needs a professional copywriter — expensive (₪1,500-4,000+ per page), slow (2-5 days), inconsistent quality
- Not scalable as SuenoHost grows and serves more clients
- Bottleneck on copy slows down the entire landing page delivery pipeline

### The Solution — AI Landing Page Copy Agent
- Customer fills out a brief in the builder (target audience, dream outcome, main pain, barriers, social proof)
- Builder sends API call with questionnaire + JSON template to the AI agent
- Agent writes copy using 60+ proven headline formulas, PAS framework, Israeli consumer psychology, emotional triggers
- Auto quality checks (no AI clichés, specific numbers, natural Hebrew, structure validation — 10 checks per output)
- Returns ready JSON → page goes live in Elementor automatically

### Key Differentiators
- **60+ proven headline formulas** tested on Hebrew landing pages
- **Israeli consumer psychology** — dugri, skepticism, specificity
- **Emotional triggers** — Loss Aversion, Curiosity Gap, Social Proof, FOMO
- **Anti-AI detection** — 13+ banned phrases, cliché detection, auto quality checks
